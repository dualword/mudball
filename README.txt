MUDBALL - 1.0 Beta

mudball.sourceforge.net

====================================================================================================
InGame Controls:

- Left/Right/Up/Down Arrows -> move
- Space -> Jump
- ESC -> Open InGame Menu

- Wiimote if available (rotate it to move and shake up to jump)

====================================================================================================
Level Editor Controls:

- Left/Right/Up/Down Arrows -> move
- Right Control -> decrese Layer
- Right Shift -> increase Layer
- A -> Add Segment
- D -> Delete Segment
- C and X -> Change Segment
- R -> Rotate Segment
- F2 -> Save level (input filename in console!)
- F3 -> Load level (input filename in console!)
- ESC -> Exit Level Editor
